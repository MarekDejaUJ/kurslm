# Dane kursowe

Ten katalog może zawierać własne korpusy prowadzącego:

- `korpus_polityczny.rds`,
- `korpus_naukowy.rds`,
- `korpus_faktograficzny.rds`,
- `respondenci_200.csv`.

Jeżeli plików nie ma, skrypty kursowe użyją wbudowanych danych przykładowych. Minimalny standard korpusu to kolumna `text`; zalecane kolumny dodatkowe to `id`, `jezyk`, `zrodlo` i `typ_korpusu`.
